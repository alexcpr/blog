Installation

1. Installer le paquet npm pretty-diff

npm install -g pretty-diff

2. Ajoutez votre nom d’utilisateur github dans la configuration globale de git

git config --global github.user "VotreNomDutilisateurGitHubIci"

3. Créez un jeton d’accès personnel (n’oubliez pas de cocher la case “Créer un gist”) afin de faire fonctionner pretty-diff. Une fois que vous avez créé le jeton, exécutez cette commande

git config --global gist-diff.token "VotreTokenPersonnelIci"

Vous êtes maintenant prêt à utiliser gist-diff!

Créer un Gist

Maintenant vous pouvez simplement exécuter la commande gist-diff afin de créer un gist avec le contenu de git diff

gist-diff

Lorsque gist-diff a créé le gist, il ouvrira l’URL du gist dans votre navigateur
